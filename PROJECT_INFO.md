# 📋 Информация о проекте

## 🎯 Описание

Viral Telegram Bot - это полнофункциональный Telegram-бот с вирусными механиками, системой монетизации и автономным привлечением пользователей.

## ✨ Ключевые особенности

### Для пользователей:
- 🎨 Генерация мемов и мотивационных цитат
- 🎮 Викторины по разным темам
- 🎲 Мини-игры (кости, камень-ножницы-бумага)
- 📊 Система уровней и опыта
- 👥 Реферальная программа с бонусами
- 💎 VIP подписка с безлимитом
- 🔮 Ежедневные предсказания

### Для владельца:
- 💰 Монетизация через VIP ($2/мес)
- 💝 Донаты через PayPal/YooMoney/QIWI
- 📈 Встроенная аналитика
- 👨‍💼 Админ-панель
- 📢 Система рассылок
- 🤖 Автоматические уведомления

### Технические:
- 🏗️ Модульная архитектура
- 🗄️ SQLite/PostgreSQL
- 🐳 Docker поддержка
- 📝 Подробное логирование
- 🔒 Защита от спама
- ⚡ Асинхронная работа

## 📊 Статистика проекта

- **Язык:** Python 3.11+
- **Фреймворк:** aiogram 3.4.1
- **База данных:** SQLite/PostgreSQL
- **Строк кода:** ~3000+
- **Файлов:** 25+
- **Время разработки:** Готов к запуску

## 🔧 Технологический стек

### Backend:
- **Python 3.11+** - основной язык
- **aiogram 3.4.1** - Telegram Bot API
- **SQLAlchemy 2.0** - ORM
- **aiosqlite/asyncpg** - асинхронные драйверы БД
- **APScheduler** - планировщик задач
- **Pillow** - обработка изображений

### DevOps:
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация
- **Git** - контроль версий

### Опционально:
- **PostgreSQL** - для масштабирования
- **Redis** - кэширование
- **Nginx** - load balancing
- **Prometheus/Grafana** - мониторинг

## 📈 Метрики и KPI

### Пользовательские метрики:
- Регистрации в день
- DAU (Daily Active Users)
- Retention (1d, 7d, 30d)
- Среднее время в боте
- Количество генераций на пользователя

### Бизнес-метрики:
- Конверсия в VIP
- MRR (Monthly Recurring Revenue)
- ARPU (Average Revenue Per User)
- Churn rate
- LTV (Lifetime Value)

### Вирусные метрики:
- K-factor (коэффициент виральности)
- Реферальная конверсия
- Sharing rate
- Viral coefficient
