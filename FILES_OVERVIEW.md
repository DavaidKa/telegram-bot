# 📁 Обзор файлов проекта

## 🎯 Основные файлы

### `main.py`
Точка входа в приложение. Инициализирует бота, регистрирует обработчики, настраивает планировщик задач.

### `config.py`
Конфигурация приложения. Загружает переменные окружения из `.env` файла.

### `requirements.txt`
Список всех Python зависимостей проекта.

---

## 🗄️ База данных (`database/`)

### `models.py`
SQLAlchemy модели:
- `User` - пользователи
- `Payment` - платежи
- `GeneratedContent` - созданный контент
- `Quiz` - викторины
- `QuizResult` - результаты викторин
- `Achievement` - достижения

### `database.py`
CRUD операции и вспомогательные функции для работы с БД.

---

## 🎮 Обработчики (`handlers/`)

### `start.py`
- Команда `/start` - регистрация и приветствие
- Главное меню
- Статистика пользователя
- Команда `/help`

### `content.py`
- Создание мемов
- Генерация контента
- Таблица лидеров
- Sharing функционал

### `games.py`
- Викторины по категориям
- Мини-игры (кости, камень-ножницы-бумага)
- Предсказания
- Система опыта

### `premium.py`
- Информация о VIP
- Покупка подписки
- Обработка платежей
- Донаты

### `referral.py`
- Реферальная программа
- Генерация реферальных ссылок
- Топ рефералов
- Бонусы за приглашения

### `admin.py`
- Админ-панель
- Статистика бота
- Рассылка сообщений
- Управление пользователями
- Выдача VIP

---

## 🔧 Сервисы (`services/`)

### `meme_generator.py`
Генерация мемов с использованием Pillow:
- Простые мемы с текстом
- Мотивационные цитаты
- Случайные мемы
- Водяные знаки

### `quiz_service.py`
Система викторин:
- Вопросы по категориям
- Проверка ответов
- Подсчет результатов
- Рейтинги

### `game_service.py`
Мини-игры:
- Игра в кости
- Камень-ножницы-бумага
- Угадай число
- Ежедневные предсказания

### `notification.py`
Система уведомлений:
- Ежедневный контент
- Викторина дня
- Предсказания
- Напоминания о VIP
