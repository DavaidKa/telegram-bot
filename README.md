## 📖 Описание

Готовый к запуску Telegram-бот с вирусными функциями (генерация мемов, AI-контент, викторины), реферальной системой и премиум-подпиской. Создан для автономного привлечения пользователей и монетизации без рекламы.

## ✨ Функционал

### Для пользователей:
- 🎨 **Генерация мемов** - создание мемов с текстом, мотивационные цитаты
- 🎮 **Викторины** - 4 категории (кино, музыка, география, наука), 15+ вопросов
- 🎲 **Мини-игры** - кости, камень-ножницы-бумага, предсказания
- 📊 **Геймификация** - система уровней, опыта и достижений
- 🏆 **Рейтинги** - таблица лидеров по опыту
- 👥 **Рефералы** - приглашай друзей и получай бонусы
- 💎 **VIP подписка** - безлимитная генерация за $2/мес

### Для владельца:
- 👨‍💼 **Админ-панель** - управление ботом и пользователями
- 📊 **Аналитика** - статистика в реальном времени
- 📢 **Рассылки** - массовые уведомления пользователям
- 💰 **Монетизация** - VIP, донаты (PayPal, YooMoney, QIWI)
- 🤖 **Автоматизация** - ежедневные уведомления и контент

## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка .env
Скопируйте `.env.example` в `.env` и заполните:
```bash
cp .env.example .env
```

### 3. Запуск
```bash
python main.py
```

### 4. Запуск с Docker
```bash
docker-compose up -d
```

## Стратегия роста без рекламы

### 1. Вирусные механики
- Генерация мемов с водяным знаком бота
- Возможность поделиться результатом в Stories
- Викторины с рейтингом друзей

### 2. Реферальная система
- 3 дня VIP за каждого приглашенного
- Топ рефералов с призами

### 3. Ежедневный контент
- Мем дня
- Викторина дня
- Push-уведомления

### 4. Геймификация
- Достижения
- Уровни пользователей
- Таблица лидеров

- Celery для фоновых задач
- Nginx + несколько инстансов бота
