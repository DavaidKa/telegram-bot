# 🚀 Запуск на Replit (САМЫЙ ПРОСТОЙ способ!)

## Запуск бота за 3 минуты БЕЗ установки чего-либо!

### Шаг 1: Создайте Repl (1 минута)
1. Перейдите на https://replit.com
2. Нажмите "Create Repl"
3. Выберите "Import from GitHub"
4. Вставьте ссылку на ваш репозиторий (или создайте новый Python Repl)

### Шаг 2: Загрузите файлы (1 минута)
Если создали пустой Repl:
1. Загрузите все файлы проекта (можно перетащить папку)
2. Или используйте Git:
   ```bash
   git clone <ваш-репозиторий> .
   ```

### Шаг 3: Настройте Secrets (1 минута)
1. В левой панели найдите "Secrets" (🔒)
2. Добавьте:
   - Key: `BOT_TOKEN`
   - Value: `8742742015:AAFTS2z3y5MSYGvhwBymmGR7h3RAKpsZ5W0`
3. Нажмите "Add Secret"

### Шаг 4: Запустите! (30 секунд)
1. Нажмите зеленую кнопку "Run"
2. Replit автоматически установит зависимости
3. Бот запустится!

### Шаг 5: Проверьте
1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте /start
4. Работает! 🎉

## 🔧 Настройка .replit файла

Создайте файл `.replit` в корне проекта:

```toml
run = "python main.py"
language = "python3"

[nix]
channel = "stable-22_11"

[deployment]
run = ["python", "main.py"]
```

## 💡 Советы

1. **Держите бота онлайн 24/7:**
   - Используйте UptimeRobot (https://uptimerobot.com)
   - Пингуйте ваш Repl каждые 5 минут

2. **Добавьте себя в админы:**
   - Узнайте свой ID: @userinfobot
   - В Secrets добавьте `ADMIN_IDS` = `ваш_id`

3. **Просмотр логов:**
   - Все логи видны в консоли Replit
   - Или откройте файл `bot.log`
